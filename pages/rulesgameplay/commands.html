Commands

General Commands

  • /calendar - displays current year, season, month
  • /money - counts they money in your inventory to display a balance

Character Commands

  • /create - creates a character
  • /delete - deletes a character
  • /character, /char - Opens character menu
  • /character firstname <name> - changes your character's first name
  • /character lastname <name> - changes your character's last name
  • /character togglelastname - toggles the display of your last name
  • /character gender <gender> - changes your character's gender
  • /character cooldown [hunt] - displays your character's deletion cooldown or, if "hunt" argument added, your hunt cooldown
  • /card [character] - displays character information
  • /bio [bio] - displays or sets character bio
  • /reroll - rerolls your character's subelements
  • /rerolls - displays character's remaining rerolls

Chat Commands

  • /msg <username> [message] - directly messages someone out-of-character
  • /r <message> - replies to someone
  • /bird <character> - directly messages someone in-character
  • Chat channel commands

Combat Commands

  • /duel <character> - displays current year, season, month
  • /dttd <character> - requests to duel someone; does not result in knock-outs
  • /hunt <character> - begin a hunt against someone

Locking Commands

  • /lock, /cprivate - Lock a door/chest
  • /lock <player> ... - Lock door/chest to you and specificed players separated by space
  • /modify <add|remove> <player> - add/remove player to access list of a protected door/chest
  • /cdonation - set chest to donation mode
  • /cpassword <password> - Password lock a door/chest
  • /cunlock <password< - Unlocks door/chest locked with password
  • /unlock - unlocks locked door/chest
  • /cpublic - protect chest/door while allowing public access

Nation Commands

  • /nations, /nation, /n
  • /n help - displays a help menu
  • /n info - displays info on whatever geofeature(s) you're in
  • /n listnations - displays list of all nations
  • /n nation, /n n
    • /n n banish <nation> <character> - banishes a character from a nation
    • /n n createcurrency <nation> <name> <material> <value> - creates a new currency for a nation
      • (requires "Settings" nation permission)
  • /n n deny <nation> - denies an inventation to a nation
  • /n n elements
  • /n n elements set <nation> <elements separated by a space> - sets a nation's allowed elements
  • /n n elements list <nation> - lists a nation's allowed elements
  • /n n info <nation> - shows info on a nation
  • /n n invite <nation> <character> - invites a character to join a nation
    • (requires "Invite" nation permission)
  • /n n invites <nation> - lists outstanding invites to a nation
    • (requires citizenship of nation)
  • /n n join <nation> - joins a nation by accepting an invitation
  • /n n leave <nation> - leaves a nation
  • /n n listnodes <nation> - lists a nation's nodes
  • /n n listprovinces <nation> - lists a nation's provinces
  • /n n listsettlements <nation> - lists a nation's settlements
  • /n n mint <nation> <currency> <amount> - mints/prints a currency
    • (requires "Mint" nation permission)
  • /n n revoke <nation> <character> - revokes an existing invitation to a nation
    • (requires "Invite" nation permission)
  • /n n setbanner <nation> <banner-id> - sets a nation's banner with a valid needcoolshoes.com banner ID
    • (require "Settings" nation permission)
  • /n n setcapital <nation> <settlement> - sets a nation's capital settlement
    • (require "Settings" nation permission)
  • /n n setcolor <nation> <color-code> - sets a nation's color by hex-code
  • /n n setcurrency <nation> <currencies separated by a space> - set a nation's currencies
    • (require "Settings" nation permission)

Province Commands

  • /n province, /n p
  • /n p cede <province> <nation> - cedes a province to another nation
    • (requires "Manage Provinces" nation permission)
  • /n p info <province> - shows info on a province
  • /n p merge <province1> <province2> <new-name> - merges two provinces
    • (requires "Manage Provinces" nation permission)
  • /n p rename <province> <new-name> - renames a province
    • (requires "Manage Provinces" nation permission)
  • /n p secede <province> - declares a province's independence from its parent nation
    • (requires province leader role)
  • /n p setbanner <province> <banner-id> - sets a province's banner
    • (requires "Settings" nation permission)

Settlement commands

  • /n settlement, /n s
  • /n s banish <settlement> <character> - banishes someone from a settlement
    • (requires "Banish" settlement permission)
  • /n s deny <settlement> - denies an invitation to a settlement
  • /n s info <settlement> - shows info on a settlement
  • /n s invite <settlement> <character> - invites someone to a settlement
    • (requires "Invite" settlement permission)
  • /n s invites <settlement> - shows outstanding invitations to a settlement
    • (requires "Invite" settlement permission)
  • /n s join <settlement> - joins a settlement as long as you have been invited to it
  • /n s leave <settlement> - leaves a settlement
  • /n s rename <settlement> <new-name> - renames a settlement
    • (requires "Manage Provinces" nation permission)
  • /n s revoke <settlement> <character> - revokes someone's invitation to a settlement
  • /n s setbanner <settlement> <banner-id> - sets a settlement's banner
    • (requires "Settings" nation permission)
  • /n s setprimary <settlement> - sets your primary settlement
  • /n s whitelist
    • /n s whitelist add <settlement> <character> - adds a character to a settlement's whitelist
      • (requires "Whitelist" nation permission)
    • /n s whitelist list <settlement> - lists characters in a settlement's whitelist
      • (requires citizenship of nation)
    • /n s whitelist remove <settlement> <character> - removes a character from a settlement's whitelist
      • (requires "Whitelist" nation permission)

Node Commands

  • /n node
  • /n node locate <node> - displays the coordinates for a node
  • /n node materials list <node> - lists the materials offered by a node
  • /n node search <material> - lists nodes containing the given material
  • /n node whitelist
    • /n node whitelist add <node> <nation> - adds a nation to a node's whitelist
      • (requires "Whitelist" nation permission)
    • /n node whitelist addall <nation> <ally-nation> - adds a nation to all of your nation's nodes
      • (requires "Whitelist" nation permission)
    • /n node whitelist list <node> - lists a node's whitelisted nations
      • requires citizenship of node's nation
    • /n node whitelist remove <node> <nation> - removes a nation from a node's whitelist
      • (requires "Whitelist" nation permission)
    • /n node whitelist removeall <nation> <other-nation> - removes <other-nation> from all of <nation>'s nodes
      • (requires "Whitelist" nation permission)

Rank Commands

Nations, Provinces, and Settlements all have individual ranking systems, controlled by their own set of subcommands. In this section, "/*" will be used as a placeholder for /n n, /n p, and /n s for nation, province, and settlement respectively as well as <*> for the name of the nation, province, and settlement respectively.

  • /* rank, /* r
  • /* r create <*> <name> <priority> - creates a new rank with name and priority (Priority ranges from 2-99 for nations, 2-79 for provinces, and 2-59 for settlements)
  • /* delete <*> <name> - deletes the rank from <*>
  • /* r list <*> - lists ranks in <*>
  • /* r permissions <*> <rank> - opens a menu to edit a rank's permission in <*>
  • /* r priority <*> <rank> <priority> - sets a rank's priority, or displays the current priority if <priority> is omitted
  • /* r rename <*> <rank> <newname> - renames a rank
  • /* r set <*> <character> - opens a menu to set a character's ranks

War Commands

  • /battle
  • /battle join - joins a battle's queue if you're standing in the battlefield
  • /battle leave - leaves a battle
