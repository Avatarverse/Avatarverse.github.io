Resource Pack

Requirement

A significant portion of the experience the server has to offer is greatly enhanced by the resource pack, therefore it is required that you load it in order to maximize your gameplay and roleplay.

How to get the resource pack

The server automatically downloads the resource pack for you (enabled by default in your minecraft setting).

If any complications occur that prevent you from properly loading the resource pack, you can download the zip file below and place it in your resource pack folder.

You should be able to open that folder from the Minecraft launcher as well.
