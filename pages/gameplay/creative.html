Creative Server

In addition to our main RPG server and combat-focused PvP server, we also offer a Creative server for you to express yourself through building, with fewer restrictions. Use /creative to join the Creative server.

The Creative server has an array of plots for you to build on. You will need to claim a plot to begin building. You can claim a random plot by using /plot auto. You can also teleport to your plot using /plot home. If you want to claim a specific plot, you can use /plot claim to claim the plot you are standing on. It's also possible to trust other players to your plot using /plot trust.
