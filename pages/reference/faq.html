Frequently Asked Questions

I'm new. What do I do?

Start by reading the Introduction page! It will give you a jumpstart on the basics and where to go from there.

How do I get food and a house?

Nation Leaders themselves, or someone they have appointed should be able to assist you with this process. It is recommended to reach out in your nation's Discord which can be found in the official Avatarverse Discord Server Guide Section.

Can I join the server on my Bedrock account?

Yes. Avatarverse is cross-platform. Learn how to join with Bedrock.

How does the bending system work?

You can learn more about the bending system in the Bending Page.

How do I roleplay?

You can familiarize yourself with the Chat System and use the roleplay channels to chat as your developed character.

How does time work in Roleplay?

1 Roleplay (RP) Year is 8 In Real Life (IRL) Weeks

Check the current RP date with /calendar

What are the restrictions on building and exploration?

As stated in RPG Server Rules #3, building structures must be consistent with the world of Avatar which is mostly East-Asian Style.

You are free to explore the world, however:

  • Some areas are restricted in Roleplay and you may face in-roleplay consequences for trespassing
  • Breaking into a locked room without a roleplay reason is considered griefing and you will be punished as such

Who is the Avatar? Can I become the Avatar?

We don't currently have an Avatar. We have had several in the past, but they have all passed on. The Avatar is a special role that is chosen by the Roleplay Council. If you are interested in becoming the Avatar, you can apply for the role when the time comes.

How can I report a player or an issue on the server?

Create a ticket on discord in the #support channel. A Staff member will process it as soon as possible.

Can I use mods or resource packs on the server?

You may use mods or resource packs as long as they don't violate Server Rules, including but not limited to allowing you:

  • To gain an unfair advantage
  • To cheat (duplicate items, creative mode, teleport, etc...)
  • To metagame (find player locations for example)
  • To use X-ray vision or similar
  • To bypass rules or staff punishments

When will Bedrock players be able to see the custom mobs?

This is a Java server with added functionality for Bedrock players to join. However, we don’t optimize around the Bedrock experience. Custom mobs use a special entity called a display entity that simply doesn’t exist on Bedrock Edition, so there is no easy way to convert it to a Bedrock screen; they instead appear invisible. The same entity is used in certain abilities, especially plantbending, and other aspects of the server. For the custom mobs, we could use an older and laggier version of the ModelEngine plugin which uses armor stands instead of the display entities, but this would result in poorer performance for the entire server. I don’t know when or if Bedrock Edition will add display entities, but I hope they do or we find a workaround. Bedrock will never have a completely seamless experience on a Java server; the fact that it’s cross-platform is already a miracle given the stark disparities between the two platforms. The best way to experience Avatarverse will always be Java Edition, so we recommend that, but we also don’t want to totally bar Bedrock users from experiencing our world.
