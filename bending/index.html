Bending

On Avatarverse, bending is the psychokinetic ability to control an element. If you've seen Avatar: the Last Airbender, you're familiar with this concept.

Each character has an element (or nonbending) and a set of abilities they can use for each element. Abilities are used by performing certain actions on different hotbar slots, onto which an ability is bound. There are also combos, which are sequences of bound abilities to produce a new ability, and passives, which are abilities that occur naturally or without an intentional activation.

Getting started

First, you need to create a character. When you finish creating a character, you will be randomly assigned subelements (or "affinities"). If you didn't get a message about affinities, you can check which ones you received with /b who <your username>. With the same command, you can check anyone's elements, subelements, and binds.

Next, you will want to know which abilities are available to you. To do this, use /b display <element>, which displays a list of abilities you can use. Make sure to check the subelements' and combos' list too.

Next, you should go through the descriptions of these abilities and select some you want to try using. Ideally, if you're new to bending, you should test out each ability and then choose the 9 that are most useful to you (9 is the number of slots in your hotbar). To see an ability's description, use /b help <ability>. It often also displays the instructions for activation/usage.

You will also need to know how to bind abilities to slots: /b bind <ability> [slot] (or instead of writing the slot, just use the command while the desired slot is selected).

Commands

  • /b bind <ability> [slot], /b b - binds an ability to a slot
  • /b clear [slot], /b c - clears all binds or just one slot
  • /b copy <username>, /b co - copies another player's binds
  • /b display <element>, /b d - displays abilities of a given element
  • /b help <topic> - provides help on an ability, element, other topic
  • /b preset, /b p - allows for creating, deleting, binding, listing presets
  • /b toggle, /b t - toggles bending or an element
  • /b who <username>, /b w - displays player element/ability info
