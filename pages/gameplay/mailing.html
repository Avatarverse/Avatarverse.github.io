Mailing

Avatarverse uses an immersive mailing system that allows players to send and receive mail in-game through use of courier mobs and personal mailboxes.

Character Commands

You can reference a list of character and mail related commands here

Mailboxes

Avatarverse makes use of a custom plugin in order to enhance the Roleplay and Gameplay experience. The mailing system provides you with the ability to send items to a specified character, assuming they also have a mailbox set up

Creating a Mailbox

  • Place a sign somewhere in a property where that character has permissions (as the owner or a trusted member)
  • On the first line, write [Mailbox]
  • On the second line, give the mailbox a name to identify it by (ex. Office Mailbox)

Sending Mail

You can send mail to another character by running the command /mail CharacterName or /mail username:CharacterName for offline characters.
