Resource pack

How to get the resource pack

Due to Minecraft's limitations, we were unable to make this Optifine resource pack automatically downloadable when you join the server. If you want to see the server's custom textures and animal models, you'll have to use both Optifine and this resource pack.

Optifine download

Go here and download the version of Optifine corresponding to the version of Minecraft that you use. Open the downloaded file and let it install. Use the new Optifine installation in your launcher.

Resource pack download

  • 1.18 resource pack download
  • 1.19 resource pack download

Go to one of the above links and download the zip file. Open your resource pack folder (you can do this from Minecraft) and place the file there. Then enable the Avatarverse resource pack.
